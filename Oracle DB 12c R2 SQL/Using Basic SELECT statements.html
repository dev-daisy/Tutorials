It quite literally is nothing, there was no value ever entered, it does not equate to anything else nor can it be compared to anything else. It quite simply is nothing. So if you try to compare nothing to something, it's just not a valid comparison, including any other null. In other words, a null does not equal another null, it's just nothing.
